Um sistema de diagnóstico é de uma maneira geral um problema adequado à utilização do paradigma lógico, uma vez que é possível inferir com precisão um diagnóstico, apenas tomando como base as repostas (ou decisões) tomadas pelo usuário. A solução implementada em Prolog utiliza uma base de dados que inclui um conjunto de doenças e perguntas que conduzem o usuário ao diagnóstico final.
No caso do Java, foi implementada a mesma solução sob uma ótica de orientação a objetos. Neste caso, como o paradigma é menos adequado para tal problema, em comparação ao Prolog, foi necessário recorrer ao modelo de algoritmo de uma árvore binária, onde cada nó representa por uma pergunta, e está relacionado a duas outras perguntas ou doenças que serão respectivamente o caminho a percorrer em uma resposta positiva ou negativa do usuário. É evidente perceber que para a modelagem em objetos, foram utilizadas por volta de 5 classes (em oposição ao Prolog, que gerou um programa bem menos extenso, se considerarmos as linhas de código).
